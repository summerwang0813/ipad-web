# DJI 官网设计风格沉淀

调研日期：2026-06-04  
对象：DJI 大疆创新中文官网首页与公共导航/页脚样式  
定位：高端消费电子、影像科技、户外与专业创作工具品牌官网

## 1. 设计关键词

- **科技克制**：以黑、白、灰为主体，界面元素很少抢产品风头。
- **产品主导**：首屏和核心区块依赖大幅真实产品图、场景图或视频感画面，而不是插画化表达。
- **轻信息、强视觉**：主标题短，副标题简洁，CTA 数量有限。
- **空间感充足**：大留白、大图、中心构图，页面节奏接近高端硬件发布页。
- **中性专业**：视觉语言不娱乐化，适合无人机、手持影像、专业影像、储能等多产品线统一呈现。
- **购买路径清晰**：导航、产品卡、CTA、页脚服务入口都围绕“了解产品 -> 比较/选择 -> 购买/支持”展开。

## 2. 品牌字体

官网 CSS 中可确认的字体栈：

```css
font-family: "Open Sans", "PingFang SC", "Microsoft YaHei", "Helvetica Neue",
  "Hiragino Sans GB", "WenQuanYi Micro Hei", Arial, sans-serif;
```

另有品牌/图标相关字体：

```css
font-family: "Dji", "PingFang SC", "Microsoft YaHei", "Helvetica Neue",
  "Hiragino Sans GB", "WenQuanYi Micro Hei", Arial, sans-serif;
font-family: "FontAwesome";
```

### 字体使用建议

| 场景 | 字体 | 字重 | 特征 |
|---|---|---:|---|
| 英文产品名 | Open Sans | 500 / 600 | 清晰、偏现代科技 |
| 中文正文 | PingFang SC / Microsoft YaHei | 400 | 中性、可读性强 |
| 大标题 | Open Sans + 中文系统字体 | 500 / 600 | 不宜过粗，保持克制 |
| 导航/按钮 | Open Sans + 中文系统字体 | 400 / 600 | 小字号、清晰扫描 |

## 3. 颜色系统

官网主色非常克制，核心是黑白灰，蓝色只作为链接或关键行动。

### 基础色

| Token | 色值 | 用途 |
|---|---|---|
| `color-black` | `#000000` | 深色背景、深色文本、遮罩 |
| `color-white` | `#FFFFFF` | 浅色背景、深色图上的文字 |
| `gray-900` | `rgba(0,0,0,.85)` | 主文本 |
| `gray-700` | `rgba(0,0,0,.65)` | 次级文本、说明 |
| `gray-500` | `rgba(0,0,0,.45)` | 辅助文本、弱化信息 |
| `gray-100` | `rgba(0,0,0,.04)` | 浅灰背景、悬浮底 |
| `gray-200` | `rgba(0,0,0,.06)` | 分割线、卡片边界 |
| `surface-light` | `#F5F5F5` / `#F7F9FA` | 产品区块浅背景 |
| `surface-dark` | `#000000` / `#303233` | 暗色沉浸区 |

### 强调色

| Token | 色值 | 用途 |
|---|---|---|
| `link-blue` | `#0070D5` | 文字链接、主要跳转 |
| `link-blue-hover` | `#2490E3` | hover 状态 |
| `link-blue-active` | `#0058B0` | active 状态 |
| `ios-blue` | `#007AFF` | 首页局部 CTA / 高亮链接 |
| `notice-orange` | `#FA8C16` | 提醒、标签、促销类提示，少量使用 |

### 色彩原则

- 页面不要做大面积彩色渐变；产品图本身承担主要视觉情绪。
- 蓝色只用于“可点击”和“需要前进”的地方。
- 暗色图上文字使用白色，透明度分层建议为 `100% / 65% / 45% / 30%`。
- 浅色页中正文首选 `rgba(0,0,0,.85)`，不要直接用纯黑铺满长文本。

## 4. 排版尺度

从官网 CSS 可观察到的常用字号/行高：

| 层级 | 桌面建议 | 行高 | 字重 |
|---|---:|---:|---:|
| Hero 标题 | 40px | 44px | 500 / 600 |
| 区块标题 | 32px | 36-40px | 500 / 600 |
| 卡片标题 | 24px | 32-36px | 500 |
| 产品副标题 | 18-20px | 24-28px | 400 / 500 |
| 正文 | 16px | 24px | 400 |
| 导航/按钮 | 14px | 20px | 400 / 600 |
| 辅助信息 | 12px | 16px | 400 |

### 排版气质

- 标题短句优先，例如“创作，尽在掌握”“航拍新旗舰”。
- 产品名通常作为视觉锚点，搭配一句功能/定位描述。
- 不使用强烈字距压缩，整体字距保持自然。
- 英文与数字信息保持清晰，例如产品型号、价格、参数、版本名。

## 5. 布局系统

### 页面结构

1. **顶部导航**
   - 左侧品牌 Logo。
   - 中间产品分类：航拍无人机、手持摄影设备、储能及家居科技、商用产品及方案、探索精彩等。
   - 右侧搜索、账号、地区/语言。
   - 导航高度紧凑，视觉层级弱于首屏内容。

2. **Hero 首屏**
   - 大幅产品/场景图或视频式背景。
   - 文案通常居中或顶部居中。
   - CTA 一般为“了解更多”“立即购买”等少量链接。
   - 首屏不放复杂表单，不做营销长文。

3. **产品推荐区**
   - 大卡片 + 双列/多列产品卡组合。
   - 每张卡有产品图、产品名、短描述、CTA。
   - 浅灰背景承载产品矩阵，视觉更像“展台”。

4. **内容/探索区**
   - 使用影像作品、场景故事、创作灵感、服务支持等入口。
   - 图片优先，文字精简。

5. **页脚**
   - 多列信息架构。
   - 包含产品分类、购买渠道、服务支持、社区/社媒、订阅、法律信息。
   - 视觉更工具化，信息密度高于主体页面。

### 栅格与间距

- 桌面端内容宽度建议：`1200px - 1440px` 居中容器。
- Hero 高度建议：`640px - 820px`，视产品图比例调整。
- 产品卡间距建议：`16px / 24px / 32px`。
- 页面区块上下间距建议：`64px - 120px`。
- 卡片内部留白建议：`24px - 40px`。

## 6. 常用组件

### 6.1 顶部导航

**形态**

- 透明/浅色/深色三种状态，根据首屏背景切换。
- 分类导航 hover 展开下拉菜单。
- 导航文字小而克制，通常 14px。

**设计要点**

- 不用厚重边框，最多使用 `rgba(0,0,0,.06)` 级别分割。
- 下拉层背景以白色或深色半透明为主。
- 产品入口可配缩略图，但不要过度装饰。

### 6.2 Hero 产品模块

**结构**

```text
背景图/视频
产品名或系列名
一句核心卖点
CTA 链接组
```

**建议规范**

- 标题：40px / 44px / 500。
- 副标题：20px / 24px / 400-500。
- CTA：14-16px，蓝色或白色文字链接。
- 深色背景上文案使用白色，副标题可用 `rgba(255,255,255,.65)`。

### 6.3 CTA 按钮/链接

官网更常见的是文字链接式 CTA，而不是大面积实心按钮。

**类型**

- 主要链接：蓝色文字，如“了解更多”。
- 次要链接：白色或黑色文字，根据背景切换。
- 购买按钮：可使用蓝色实心或文字链接，具体看页面上下文。

**状态**

| 状态 | 样式 |
|---|---|
| Default | `#0070D5` |
| Hover | `#2490E3` |
| Active | `#0058B0` |
| Disabled | `rgba(0,0,0,.25)` |

### 6.4 产品卡片

**结构**

```text
产品图
产品名
一句短描述/价格/标签
了解更多 / 购买
```

**样式**

- 背景：白色或 `#F7F9FA`。
- 圆角：官网整体偏克制，建议 `0-8px`。
- 阴影：尽量不用；需要层级时使用极轻阴影或分割线。
- 图片：产品必须清晰可识别，避免抽象氛围图。

### 6.5 产品分类入口

**适用场景**

- 航拍无人机、手持摄影设备、商用方案、配件、服务支持。

**样式**

- 使用图标/产品缩略图 + 分类名称。
- 分类文案不超过 8 个中文字符为佳。
- 鼠标悬浮时可轻微上移、变亮或显示二级菜单。

### 6.6 规格/对比表

**风格**

- 信息密集但视觉克制。
- 使用细分割线、浅灰底、深灰文字。
- 参数名弱化，参数值突出。

**建议**

- 表头固定产品名和产品图。
- 关键差异用蓝色或小标签标注。
- 不要使用过多彩色状态。

### 6.7 弹窗/地区选择

**结构**

- 标题：国家/地区选择、登录、订阅等。
- 内容：列表或表单。
- 关闭按钮：右上角图标。

**样式**

- 白底，文字 `rgba(0,0,0,.85)`。
- 分组标题弱化，列表项 hover 使用浅灰背景。
- 弹窗遮罩可使用 `rgba(0,0,0,.5)`。

### 6.8 页脚

**结构**

- 多列链接。
- 订阅输入框。
- 社交媒体入口。
- 版权、备案、政策链接。

**样式**

- 深色页脚适合品牌官网，浅色页脚适合商城/支持页面。
- 文字层级用灰度透明度区分，不靠大字号。
- 链接 hover 轻微提亮。

## 7. 图像与媒体风格

- 必须使用真实产品、真实使用场景或高质量渲染图。
- 产品通常居中、低角度或漂浮式展示，背景干净。
- 消费级产品偏生活方式：旅行、户外、城市、创作。
- 专业产品偏工业/片场/任务场景：强调可靠、精准、效率。
- 图片光线干净，避免杂乱生活感。
- 产品图应可清楚看见形态、镜头、云台、桨叶、屏幕或核心卖点。

## 8. 动效与交互

- 动效轻，主要用于 hover、菜单展开、轮播、视频背景。
- 产品卡 hover 可轻微放大图片或提升亮度。
- 下拉菜单需要快速、稳定，不应有夸张弹性。
- 滚动叙事适合新品发布页，但官网首页更偏直接陈列。
- 动效时长建议：`160ms - 300ms`。
- 缓动建议：`ease-out` 或 `cubic-bezier(.2, 0, .2, 1)`。

## 9. 内容语气

### 文案原则

- 少解释，多定调。
- 产品名优先，卖点其次。
- 避免长段落，避免情绪化夸张词。
- 使用“专业、创作、影像、轻便、旗舰、全能、掌控、探索”等关键词。

### 示例句式

- `DJI Mavic 4 Pro`
- `影像旗舰，创作进阶`
- `轻巧随行，灵感即刻启程`
- `了解更多`
- `立即购买`
- `对比机型`

## 12. 设计检查清单

- 首屏是否一眼能看到真实产品或真实场景？
- 页面是否保持黑白灰为主体，蓝色只用于行动？
- 标题是否足够短，是否避免了说明书式文案？
- CTA 是否少而清晰？
- 产品图是否比 UI 装饰更重要？
- 卡片是否克制，避免厚重阴影和过圆圆角？
- 导航是否紧凑，且不会压过首屏视觉？
- 页脚是否能承载完整产品、购买、服务、法律入口？
- 移动端是否仍保持大图、短标题、少 CTA 的节奏？
